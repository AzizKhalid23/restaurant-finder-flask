# 🍽️ Restaurant Finder (Flask + MongoDB)

This project is a simple Flask web app that allows users to sign up, log in, and search for restaurants using data stored in MongoDB.

> 📝 **This project was developed as part of the Database 2 course at Uniersity of Jeddah.**

---

## 📌 Features

- ✅ User authentication (signup & login) using SQLite
- 🔎 Restaurant search with MongoDB
- 🔍 Search by:
  - Street
  - Borough
  - Cuisine
  - Grade
  - Name
- 👤 Secure user sessions

---

## 🛠️ Technologies Used

- Python 3
- Flask
- SQLite (for user accounts)
- MongoDB (for restaurant data)
- PyMongo
- HTML/CSS (Jinja templates)
